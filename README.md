# mart-node-express

โปรเจกต์ตัวอย่าง Backend ด้วย Node.js และ Express  
ใช้สำหรับฝึกสร้าง REST API และโครงสร้าง server พื้นฐาน  
เหมาะสำหรับนักเรียน/นักศึกษาที่เริ่มต้นพัฒนา backend  
สามารถนำไปต่อยอดเชื่อมต่อฐานข้อมูลได้  
เป็นโปรเจกต์ฝึกงานและงานส่งในรายวิชาโอเพนซอร์ส
